# ONDO
OndoChain by Ondo Finance is a **Layer 1 proof-of-stake blockchain** purpose-built to **accelerate the creation of institutional-grade financial markets** onchain. By combining the openness of public blockchains with the compliance and security features of permissioned chains, Ondo Chain provides the **infrastructure to enable tokenized real-world assets to be used at scale**.
