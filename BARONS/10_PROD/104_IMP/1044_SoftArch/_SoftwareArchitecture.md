# Folder for Software Architecture Artefacts

This `SoftArchitecture`-Folder contains all artefacts that will ouline, describe and define the various **components, modules, packages etc. that together will constitute the overall software architecture**. 

Please note that this folder contains only abstract concepts and pseudocode whereas the SourceCode will reside in the `40_DEV` folder. 

Also note that these Specs here are for internal use only whereas the final deliverables for the end-user and customers will be compiled in the `11_MANUAL` folder.

---

# Verzeichnis für Software Architecture Artefakte

Dieses `Software-Architecture`-Verzeichnis enthält alle im Rahmen des Projekts erstellten Artefakte welche die Software Architektur spezifizieren und deren Betrieb dokumentieren. 

Insbesondere sind dies Softwarekomponenten, Module, Packages, IDE-Strukturen und Konfigurationsdateien, Logs etc. welche alle zusammen Teile des final ausgerollten Systems bilden.

Innerhalt der Architektur wird zwischen den finalen Produkt und der für die Implementierung verwendete Entwicklungsumgebung unterschieden. 

Man beachte dass dieses Verzeichnis lediglich abstrakte Konzepte, Manuals und Pseudocode enthält währenddem der SourceCode im `40_DEV` Unterverzeichnis liegt. 

<span style="color:red; font-weight:bold">ACHTUNG</span>: die in diesem Verzeichnis enthaltenen Dokumente sind `for internal use only` währenddem die finalen Admin und User Dokumente für Benutzer im separaten `11_MANUAL` Verzeichnis abgelegt werden!
