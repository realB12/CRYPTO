# CoinBase Transaktion

->  [Mining](../M/Mining.md) // [Mining Pool](../M/MiningPool.md)


Die Coinbase-Transaktion ist also die erste Transaktion in jedem Block und dient dazu, die Mining-Belohnung und Transaktionsgebühren an den Miner auszuzahlen und kann in einem Block so aussehen:

* **Input**: Spezieller Wert (z. B. Nonce, Nachrichten).
* **Output**: Auszahlung an die Adresse des Miners.

 Sie hat keine Eingänge, sondern generiert neue Bitcoins, und kann von Minern angepasst werden, um den Hashing-Prozess zu unterstützen.
